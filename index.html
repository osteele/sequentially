Sequentially

Sequentially is a library of temporal adverbs for JavaScript. It provides methods on a function that queue it for deferred or repeated execution, and control the rate or number of times that a function can be called. Sequentially also contains implementations of Haskell MVars and pi calculus named channels, that can be used to coordinate multiple processes.
