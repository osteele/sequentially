Sequentially

header | announcement | download: compressed (<2K min gz) | source

Sequentially is a library of temporal adverbs for JavaScript. It provides methods to queue a function for deferred or repeated execution, and to control the rate or count at which a function can be called.

Sequentially also contains implementations of Haskell MVars and pi-calculus named channels, that can each be used to coordinate data flow among asynchronously-scheduled functions.
