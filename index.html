Sequentially: Frequency Adverbs for JavaScript

header | announcement | download: compressed (<2K min gz) | source

Sequentially is a library of temporal and frequency adverbs for JavaScript. It provides methods to queue a function for deferred or repeated execution, and to control the rate or count at which a function can be called.

Sequentially also contains implementations of Haskell MVars, that can each be used to coordinate data flow among asynchronously-scheduled functions.

Functionally, Sequentially is like a blend of streams and generators, with processes. Computationally, it's similar to the π-calculus. Stylistically, it can be used in a adverbial programming, a paradigm that complements procedural (verbal), object-oriented (noun phrase), and functional (gerundive) programming.

This is a preview release. It hasn't been thoroughly tested, and I plan to make some major changes to the API as I continue to experiment how best to use this in my own programs.
